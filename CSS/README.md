1. Texto e Cores
color: red; — Altera a cor do texto (aceita nome, rgb(), ou #Hexadecimal).

font-size: 16px; — Altera o tamanho da fonte.

font-family: Arial, sans-serif; — Define a fonte do texto.

text-align: center; — Alinha o texto (left, right, center, justify).

font-weight: bold; — Define a espessura da fonte (negrito).

2. Cores de Fundo e Bordas
background-color: blue; — Altera a cor de fundo do elemento.

border: 2px solid black; — Cria uma borda (espessura, estilo e cor).

border-radius: 8px; — Arredonda os cantos do elemento (cria círculos se for 50%).

3. O Box Model (Espaçamento e Tamanho)
width: 300px; — Define a largura do elemento.

height: 150px; — Define a altura do elemento.

padding: 20px; — Espaçamento interno (entre o conteúdo e a borda).

margin: 15px; — Espaçamento externo (afasta um elemento do outro).

4. Posicionamento e Layout (Básico)
display: block; — Faz o elemento ocupar a linha toda.

display: inline; — Faz o elemento ocupar apenas o espaço do seu conteúdo.

display: flex; — Ativa o Modo Flexbox (excelente para alinhar elementos lado a lado ou um embaixo do outro).

justify-content: center; — Alinha elementos horizontalmente dentro do Flexbox.

align-items: center; — Alinha elementos verticalmente dentro do Flexbox.
